How to repair OMSI simulator files without using "Verify Integrity of Game Files"?

You can use SteamCMD to repair files without integrity checking.

  1. Downloading SteamCMD
    To download SteamCMD visit the official website: https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip (for Windows only)
     
  2. Downloading game files using SteamCMD
    To download the simulator files you need:
    - appid of the game (for OMSI 2 it is 252530).
    - your login.
    - your password.
    - Your Steam Guard Code (optional).

    Open a command prompt in the SteamCMD folder and type:
    steamcmd +login your_login +app_update appid +quit

    After logging in, the download will start.
    Attention! If this is your first time starting SteamCMD, the client update will begin and you will be prompted for your Steam Guard Code.
     
  3. Downloading add-on files using SteamCMD
    To download the simulator files you need:
    - appid of the game (for OMSI 2 it is 252530).
    - depotid of the add-on.
    - your login.
    - your password.
    - Your Steam Guard Code (optional).

    Depotid of the add-on can be found in the address bar of your browser.
    Open the Steam store page, find your add-on and write down the specific number.



    Open a command prompt in the SteamCMD folder and type:
    steamcmd +login your_login +download_depot appid depotid +quit

    After logging in, the download will start.
    Attention! If this is your first time starting SteamCMD, the client update will begin and you will be prompted for your Steam Guard Code.
     
  4. Copying files
    You can find the files to be copied in the steamapps folder.
     
  5. More information:
    https://developer.valvesoftware.com/wiki/SteamCMD
    https://developer.valvesoftware.com/wiki/Command_Line_Options#SteamCMD

        Error! App 'xxx' state is 0x212 after update job.
    This means that you do not have enough disk space.
